## 2026-04-19 - Adding aria-labels to icon-only buttons\n**Learning:** When adding ARIA labels to a single-file Vanilla JS application where components are rendered dynamically via template literals (like the Favorite Toggle Button in this project), the labels must also be injected into those string templates to ensure accessibility remains intact after dynamic rendering.\n**Action:** Always scan both the static HTML elements and the JavaScript dynamic rendering logic to ensure accessibility attributes are applied universally.
